## Health Informatics Projects

This repository contains coursework and projects related to health informatics, biomedical data analysis, clinical data systems, and genomics. The projects combine healthcare-focused computational methods with practical data analysis and machine learning workflows. 

## Homework 1: Healthcare Data Quality
**Folder:** `homework-1-healthcare-data-quality`

Topics explored:
- Healthcare data preprocessing
- Regression analysis
- Synthetic patient datasets
- Data cleaning
- Exploratory data analysis
- Statistical modeling

## Homework 2: FHIR and EHR APIs
**Folder:** `homework-2-fhir-and-ehr-apis`

Topics explored:
- FHIR resources
- Electronic Health Records (EHRs)
- Healthcare interoperability
- Clinical APIs
- Patient observations
- JSON healthcare data structures

## Homework 3: Clinical Text Deidentification
**Folder:** `homework-3-clinical-text-deidentification`

Topics explored:
- Clinical text processing
- Protected Health Information (PHI) removal
- Healthcare privacy
- Deidentification pipelines
- PyDeid
- NLP for clinical documents

## Homework 4: Clinical Vital Sign Analysis
